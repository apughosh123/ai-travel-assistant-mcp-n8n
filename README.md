# AI Travel Assistant using MCP in n8n

## Overview

This project is an AI-powered Travel Assistant built with n8n using the Model Context Protocol (MCP). The AI Agent retrieves real-time travel information from external tools instead of relying only on the language model.

## Features

- Real-time weather
- Hotel search
- Restaurant search
- Tourist attractions
- Currency exchange
- AI-generated travel itinerary
- Memory for user preferences

## Technologies

- n8n
- MCP (Model Context Protocol)
- Groq LLM
- Open-Meteo API
- OpenStreetMap Nominatim API
- Exchange Rate API

## MCP Tools

- Get Coordinates
- Get Current Weather
- Find Hotels
- Find Restaurants
- Find Attractions
- Get Exchange Rate

## Memory

The workflow includes Simple Memory to remember user travel preferences such as budget, hotel type, transportation, food preference, and activities.
